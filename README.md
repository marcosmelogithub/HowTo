# Projeto HowTo
---
O objetivo do **Projeto HowTo** é criar arquivos HowTo com a linguagem markdown, principalmente focado em *Quick Guide* para servir como uma referência para consultas rápidas e objetivas!

E para facilitar a listagem dos arquivos HowTo e a visualização do conteúdo de cada arquivo .md, foi escrito uma pequena e simples aplicação html/js que se comporta da seguinte forma:

1. Ao abrir ou atualizar a página web, são lidos todos os arquivos HowTo.md que existirem na pasta desse repositório e listados com um hyperlink de acesso para cada um.
2. Quando clicar em algum item da lista de arquivos HowTo.md, logo abaixo, na mesma página web, é exibido o conteúdo do arquivo, respeitando as marcações markdown e mermaid (diagramas)
3. Para acessar a aplicação basta clicar [Browsing Howto's](https://marcosmelogithub.github.io/ProjetoHowTo/)
4. As vantagens de usar essa aplicação web:
4.1. Não é necessário ir no repositório do github
4.2. A utilização é intuitiva como se está acostumado no dia-a-dia com aplicações web
4.3. Pode ser acessado mesmo por quem que nem tem uma conta no github
